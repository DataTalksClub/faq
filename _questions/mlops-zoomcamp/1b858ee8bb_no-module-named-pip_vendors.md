During scikit-learn installation via the command:

pipenv install scikit-learn==1.0.2

The following error is raised:

ModuleNotFoundError: No module named 'pip._vendor.six'

Then, one should:

sudo apt install python-six

pipenv --rm

pipenv install scikit-learn==1.0.2
