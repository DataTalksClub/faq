Incorrect:

df['DOlocationID'] = pd.to_numeric(df['DOlocationID'], downcast=integer) or

df['DOlocationID'] = df['DOlocationID'].astype(int)

Correct:

df['DOlocationID'] = df['DOlocationID'].astype('Int64')
